Assignment Instructions

Objectives: You understood and are able to work with Service Workers. You will also be able to demonstrate what you understood by PWA's and even create a basic PWA with the manifest file so users will be able to see the possibility of installing your web app on their desktop.

You will find in the hw4 folder of the files you downloaded, two images that you will be using to complete this part of the assignment.

You will create a simple page (named index.html) that will have a heading, and at least two paragraphs where you will be explaining what are the main characteristics of a PWA (Progressive Web Application). You will also give to the user a drop-down select so the user can select the type of background should be applied to the page - light blue or light gold - once the user clicks the button to change the background, you will use the respective images that you found in hw4 folder.

Create some style to that page using an external CSS file and start the page with a plain white background color.

Create a manifest file for that basic PWA pop-up window Chrome to install or not PWAso the user will be able to see the option to install the application when using the menu of the browser - see the image shown here when a simple PWA was created with the manifest file showing the confirmation to install or not the app (the black spot is just not to show the server that this is coming from as your would be showing the server that your application is coming from.

You will also create a service-worker.js file in order to make that PWA page work even when the user is offline - even the change of the background image should be able to work as the images would also be cached!

When you finish your work, you will upload ALL files to your web server account. Do not forget to test in your desktop/laptop browser and also in some mobile devices using https:// in front of the URL (make sure that the server you are using allows you to use https://).
